GRAIL/ISO
LIVE — PRIVATE ALPHA
The core platform. Demand-first sports card marketplace. Buyers broadcast wants (ISO = In Search Of). Sellers respond to demand. Seller inventory never publicly visible. Buyer intent data = unique monetizable asset. isosandbox.com
ISO/VAULT
LIVE
Seller inventory + collection manager. Two vault paths. Sport filter chips. Hero card + favorites rail. Slab Lab CTA. vault_entries table. My Vault is landing page on login.
ISO/SLAB
LIVE
Slab Lab. 8 colors × 3 finishes = 24 combos. GRAILISO_slab.png base64 overlay (336×485px). 3D slab unit — card locked inside, entire slab rotates. Matte / Crystal (default) / Vivid finish system.
ISO/SCAN
LIVE (Basic)
Card catalog + browse. ISOVault: 4-row pill nav, sport-scoped player search, 5 catalog tables. Pokémon (20,034 cards), NBA (3,348), Baseball (1,000+), Topps B&W (316). AI recognition: planned.
ISO/CREDITS
LIVE
Posting credit system. $1 per ISO post. Balance in profiles.credits. ISOCredits refund on grade fail. Money stays in ecosystem. Buy prompt from My Vault credits chip.
ISO/FOMO ENGINE
POST-STABILITY — DO NOT BUILD YET
The revenue engine. The nervous system. Connects every signal. Fires the right action at the right user at the right moment. Real data only. 5 trigger stacks: Supply, Demand, Market, Collection, Timing. Admin dashboard at /admin/fomo.
ISO/GRADING
SPEC LOCKED — NOT BUILT
1,000-point grading system (÷100 = X.XX). 5 categories: Centering (250), Corners (250), Edges (200), Surface (200) — all AI. Card Appeal (100) — human. 4 era tolerance bands. $30/card · $20 bulk (20+ cards).
ISO/BIOMETRICS
SPEC LOCKED — HARDWARE PENDING
4-layer card fingerprint authentication: paper fiber · print dot cluster · surface micro-scratches · edge fiber tear. 1–1,000 match confidence. 950+ = confirmed. 750–949 = human review. <750 = void.
ISO/DEFENDER
PLANNED
Fraud intelligence system. Works in tandem with ISO/BIOMETRICS and ISO/GRADING to protect both buyer and seller in every transaction. ToS: "Unable to authenticate" — not an accusation.
ISO/MARKET
PLANNED
Secondary marketplace. ISO matches convert to transactions. Escrow-based settlement. Stripe-powered. Requires attorney engagement + money transmitter license + TN Foreign LLC.
ISO/TRACK
PLANNED
Provenance + chain of custody. Every authenticated card carries a full ownership history. Pairs with ISO/BIOMETRICS fingerprint. Foundation for long-term card value credibility.
ISO/RANK
PLANNED
Seller reputation system. Leaderboard panel built. Full ranking engine pending. Disintermediation defense: ToS prohibits real-handle usernames. Value lock + rep system as primary defense.
ISO/DEMAND
PLANNED
Demand layer licensing business. Buyer intent data is the unique monetizable B2B asset. License aggregated demand signals to card shops, distributors, manufacturers. Core VC pitch data moat.
ISO/INFO
PARKED
Contextual card/player/set storytelling layer. Wikipedia API + Claude narrative inline on card detail. Seeds emotional connection to cards. No build date set.
ISO/BAY
PARKED — POST-FUNDING
eBay seller CRM. Architecture: OAuth 2.0 via Netlify Functions, eBay Sell APIs, Supabase token storage. Strategy: Embrace → Extend → Extinguish. Parked pending attorney + funding.
ISO/MOMENTUM
ATTORNEY REQUIRED
Performance-based market sentiment modifier. 🔥 badge on ISOVault tiles post player performance. Decays 24–48hr. Informational only — NOT financial advice. Required ToS language locked. Gate: attorney coverage.
ISO/GRAIL
RESERVED
Reserved name. Potential grading spinoff brand. Concept: standalone grading service built on ISO/GRADING system. No build scope set.
ISO/KIDS
ATTORNEY + COPPA
Kids-only virtual trading platform. No money — ISO credits only. Full COPPA compliance required. Attorney coverage required before any code written.
